# microbit-digital-read

Extension MakeCode untuk membaca data digital pada micro:bit.

## Fitur
- Membaca nilai digital dari pin tertentu.

## Cara Penggunaan
1. Tambahkan extension ini ke proyek MakeCode Anda.
2. Gunakan blok "Baca Digital" untuk membaca nilai digital dari pin yang diinginkan.

# microbit-ssd1315-oled

Extension MakeCode untuk menampilkan teks pada OLED SSD1315 dengan micro:bit.

## Fitur
- Inisialisasi SSD1315
- Menampilkan teks pada posisi tertentu

## Cara Penggunaan
1. Tambahkan extension ini ke proyek MakeCode Anda.
2. Gunakan blok untuk inisialisasi dan menampilkan teks.
